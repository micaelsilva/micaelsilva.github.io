Instalando minidlna no MacOS via Homebrew

Primeiro acesse o Terminal do MacOS. Uma forma rápida de chegar até lá é usando a opção Command+Espaço para usar o Spotlight, e digitar 'terminal' e enter

Na sequência instale o Homebrew, que pode ser feito através de uma única linha, que pode ser vista abaixo ou direto no site brew.sh

/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

O processo vai gerar bastante texto, pedir a sua senha de usuário do Mac e até instalar um extra chamado Command Line Tools. Tudo isso é esperado.

Assim que concluído você usará o comando

brew install minidlna

para invocar a instalação do minidlna. Ele vai causar a instalação de varias dependências (softwares externos que ele depende para fazer outras tarefas) e isso pode demorar mais ou menos dependendo do seu computador. Paciência.

Assim que ver esta mensagem é sinal que a instalação terminou com sucesso:

Ela indica como se procede daqui em diante para configurar, mas replico aqui abaixo e explico o que cada uma faz:

mkdir -p ~/.config/minidlna
cp /usr/local/opt/minidlna/share/minidlna/minidlna.conf ~/.config/minidlna/minidlna.conf
ln -s ~/Movies ~/.config/minidlna/media
brew services start minidlna

A primeira linha cria a pasta onde vai ficar a configuração do minidlna, dentro da sua pasta de usuário. O ~ (til) em sistemas como Linux e MacOS significa a pasta do usuário, o que serve de um atalho pra nao ter que escreve-la especificamente. Você pode não ver essa pasta posteriormente também porque tudo que começa com . (ponto final) nesses sistemas fica automaticamente oculto. A segunda linha copia um modelo de configuração já pronto direto da instalação do minidlna para facilitar, que já aponta lá dentro para uma possivel pasta de mídias. Aí na terceira linha o sistema cria um link entre uma pasta que exista (no caso ~/Movies seria a pasta de filmes na sua pasta de usuário, independente do idioma do sistema) e cria um atalho lá A última linha faz o Homebrew iniciar o minidlna como um serviço (programa que roda em segundo plano no sistema) e faz com que ele inicie toda vez que o computador ligar.

Se tudo der certo voce já pode abrir a TV na seção feita para ver dispositivos DLNA ou em apps como o VLC para iOS que estará aparecendo Mac DLNA Server
